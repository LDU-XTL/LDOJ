</p><div class="ptx" lang="en-US">Two positive integers are said to be relatively prime to each other if the Great Common Divisor (GCD) is 1. For instance, 1, 3, 5, 7, 9...are all relatively prime to 2006.
<br>
<br>Now your job is easy: for the given integer m, find the K-th element which is relatively prime to m when these elements are sorted in ascending order.
<br></div><p class="pst">Input</p><div class="ptx" lang="en-US">The input contains multiple test cases. For each test case, it contains two integers m (1 <= m <= 1000000), K (1 <= K <= 100000000). </div><p class="pst">Output</p><div class="ptx" lang="en-US">Output the K-th element in a single line.</div><p class="pst">Sample Input</p><pre class="sio">2006 1
2006 2
2006 3
</pre><p class="pst">Sample Output</p><pre class="sio">1
3
5

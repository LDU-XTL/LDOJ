</p><div class="ptx" lang="en-US">Let us consider the set of integer numbers between 1 and N inclusive. Let us order them lexicographically (i. e. like in the vocabulary), for example, for N = 11 the order would be: 1, 10, 11, 2, 3, 4, 5, 6, 7, 8, 9.
<br>
<br>Let us denote the position of the number K in this ordering as Q<sub>N,K</sub>. For example, Q<sub>11,2</sub> = 4. Given numbers K and M find the smallest N such that Q<sub>N,K</sub> = M.
<br></div><p class="pst">Input</p><div class="ptx" lang="en-US">Input contains two integer numbers K and M (1 <= K, M <= 10<sup>9</sup>) separated by a space.</div><p class="pst">Output</p><div class="ptx" lang="en-US">If such N that Q<sub>N,K</sub> = M exists then write to the output the smallest such N, otherwise write 0.</div><p class="pst">Sample Input</p><pre class="sio">100000001 1000000000
</pre><p class="pst">Sample Output</p><pre class="sio">100000000888888879

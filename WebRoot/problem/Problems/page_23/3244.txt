</p><div class="ptx" lang="en-US"><p>For every pair of triplets, <i>T<sub>a </sub></i>= (<i>I<sub>a</sub></i>, <i>J<sub>a</sub></i>, <i>K<sub>a</sub></i>) and <i>T</i><sub><i>b</i> </sub>= (<i>I<sub>b</sub></i>, <i>J<sub>b</sub></i>,<i> K<sub>b</sub></i>), we define the <i>difference value</i> between <i>T<sub>a</sub></i> and <i>T</i><sub><i>b</i> </sub>as follows:</span></p><p><p align="center"><i>D</i>(<i>T<sub>a</sub></i>,<i> T<sub>b</sub></i>) = max {<i>I<sub>a</sub></i> &minus; <i> I<sub>b</sub></i>, <i>J<sub>a</sub></i> &minus; <i>J<sub>b</sub></i>, <i>K<sub>a</sub></i> &minus; <i>K<sub>b</sub></i>} &minus; min {<i>I<sub>a</sub></i> &minus; <i> I<sub>b</sub></i>, <i>J<sub>a</sub></i> &minus; <i>J<sub>b</sub></i>, <i>K<sub>a</sub></i> &minus; <i>K<sub>b</sub></i>}<p></p>Now you are given <i>N</i> triplets, could you write a program to calculate the sum of the difference values between every unordered pair of triplets?</div><p class="pst">Input</p><div class="ptx" lang="en-US">The input consists of several test cases.

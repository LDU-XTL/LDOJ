</p><div class="ptx" lang="en-US"><p>WFF 'N PROOF is a logic game played with dice. Each die has six faces representing some subset of the possible symbols K, A, N, C, E, p, q, r, s, t. For example, <i>ApNp</i> is a tautology because it is true regardless of the value of <i>p</i>. On the other hand, <i>ApNq</i> is not, because it has the value 0 for <i>p=0, q=1</i>. </p><p>You must determine whether or not a WFF is a tautology. </p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>Input consists of several test cases.

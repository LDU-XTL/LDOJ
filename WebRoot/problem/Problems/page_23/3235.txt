</p><div class="ptx" lang="en-US"><span lang="en-us"><p>A super knight moves in the infinite <i>n</i>-dimensional space. Each move it can perform is described by a vector – a vector (<i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, …, <i>a<sub>n</sub></i>) indicates that a move from the square (with coordinates) (<i>x</i><sub>1</sub>, <i>x</i><sub>2</sub>, …, <i>x<sub>n</sub></i>) to the square (<i>x</i><sub>1</sub> + <i>a</i><sub>1</sub>, <i>x</i><sub>2</sub> + <i>a</i><sub>2</sub>, …, <i>x<sub>n</sub></i> + <i>a<sub>n</sub></i>) or (<i>x</i><sub>1</sub> &minus; <i>a</i><sub>1</sub>, <i>x</i><sub>2</sub> &minus; <i>a</i><sub>2</sub>, …, <i>x<sub>n</sub></i> &minus; <i>a<sub>n</sub></i>) is possible. Each knight has a prescribed set of such vectors, describing the moves this knight can make. For each knight we assume that this knight can reach anywhere in the space if it is allowed (but actually disallowed) to move along a fractional part of a vector.</p><p>We say two knights are equivalent, if they can reach exactly the same squares starting from the square (0, 0, …, 0) (by making many moves, perhaps). (Let us point out that equivalent knights may reach these squares in different number of moves). It can be shown that for every knight there exists an equivalent one whose moves are described by only <i>n</i> vectors.</p><p>Given a set of&nbsp; <i>m</i> (<i>m</i> &gt; <i>n</i>) vectors describing the moves of a super knight, determine an equivalent knight as mentioned above.</p></span></div><p class="pst">Input</p><div class="ptx" lang="en-US"><span lang="en-us"><p>The input contains exactly one test case. The first line of input contains two integers <i>m</i> and <i>n</i> (2 &le; <i>n</i> &lt; <i>m</i> &le; 100, <i>n</i> &le; 10, <i>n</i> &middot; <i>m</i> &le; 200). The next <i>m</i> lines each contains an integral <i>n</i>-dimensional vector (<i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, &hellip;, <i>a<sub>n</sub></i>). It is guaranteed that for all <i>i</i> (1 &le; <i>i</i> &le; n) if <i>n</i> = 2, |<i>a<sub>i</sub></i>| &le; 10<sup>3</sup>, otherwise |<i>a<sub>i</sub></i>| &le; 10<sup>2</sup>.</p></span></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>Output <i>n</i> <i>n</i>-dimensional vectors describing the moves of an equivalent knight, each on a separate line.</p></div><p class="pst">Sample Input</p><pre class="sio">3 2
1 0
0 5
0 7</pre><p class="pst">Sample Output</p><pre class="sio">1 0
0 1</pre><p class="pst">Hint</p><div class="ptx" lang="en-US"><p>Do not output any number longer than 50 digits. The test cases are designed that to each one there is a solution not involving any number exceeding 10<sup>20</sup> by magnitude.</p></div><p class="pst">

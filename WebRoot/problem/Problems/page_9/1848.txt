<br>N
<br>x(1) y(1)
<br>x(2) y(2)
<br>...
<br>x(N-1) y(n-1)
<br>N (3 <= N <=100) is the number of vertices. x(i) and y(i) (x(i), y(i) are integers, 1 <= x(i), y(i) <= N) represent the two vertices connected by the i-th edge.
<br></div><p class="pst">Output</p><div class="ptx" lang="en-US">The output will contain the value -1 if the problem doesn't have a solution, otherwise an integer, representing the number of added edges.</div><p class="pst">Sample Input</p><pre class="sio">7

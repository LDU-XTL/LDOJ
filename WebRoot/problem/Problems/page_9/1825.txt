</p><div class="ptx" lang="en-US">Consider m natural numbers n1, n2, ..., nm with the property n1 >= n2 >= ... >= nm > 0. 
<br>
<br>We define a Young table as an arrangement in a table of n1 + n2 + ... + nm natural numbers (bigger than 0 and any two different), so that the ith line has ni elements (1 <= i <= m) in ascending order from left to right, and the elements from the same column are in ascending order from bottom to top. 
<br>
<br>An example of Young table for m = 4, n1 = 6, n2 = 4, n3 = 4, n4 = 1 is the following:
<br><center><pre>1	2	5	9	10	15
<br>3	6	7	13		  
<br>4	8	12	14		  
<br>11					  </pre></center>
<br>Given n1, n2, ..., nm determine the number of Young tables containing the elements 1, 2, ..., n1+n2+...+nm.
<br></div><p class="pst">Input</p><div class="ptx" lang="en-US">The input has the stucture:
<br>
<br>on the first line is: the natural number m (1 <= m <= 20); 
<br>on the second line are: the numbers n1, n2, ..., nm separated by a space (n1 <= 12). </div><p class="pst">Output</p><div class="ptx" lang="en-US">The output will contain the number of Young tables that can be built.</div><p class="pst">Sample Input</p><pre class="sio">2

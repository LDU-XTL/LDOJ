</p><div class="ptx" lang="en-US"><div><p>In an edge-weighted tree, the xor-length of a path <i>p</i> is defined as the xor sum of the weights of edges on <i>p</i>:</p><center><img src="formula?tex=_%7Bxor%7Dlength%28p%29%3D%5Coplus_%7Be+%5Cin+p%7Dw%28e%29" alt="_{xor}length(p)=\oplus_{e \in p}w(e)" align="absmiddle"></center><p>⊕ is the xor operator.</p><p>We say a path the xor-longest path if it has the largest xor-length. Given an edge-weighted tree with n nodes, can you find the xor-longest path? 　</p></div></div><p class="pst">Input</p><div class="ptx" lang="en-US"><div><p>The input contains several test cases.

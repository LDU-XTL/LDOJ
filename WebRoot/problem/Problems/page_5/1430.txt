</p><div class="ptx" lang="en-US">The Stirling number of the second kind S(n, m) stands for the number of ways to partition a set of n things into m nonempty subsets. For example, there are seven ways to split a four-element set into two parts:
<br>
<br><pre>{1, 2, 3} U {4}, {1, 2, 4} U {3}, {1, 3, 4} U {2}, {2, 3, 4} U {1}
<br>{1, 2} U {3, 4}, {1, 3} U {2, 4}, {1, 4} U {2, 3}.</pre>
<br>
<br>There is a recurrence which allows to compute S(n, m) for all m and n.
<br><pre>
<br>S(0, 0) = 1; S(n, 0) = 0 for n > 0; S(0, m) = 0 for m > 0;
<br>S(n, m) = m S(n - 1, m) + S(n - 1, m - 1), for n, m > 0.</pre>
<br>
<br>Your task is much "easier". Given integers n and m satisfying 1 <= m <= n, compute the parity of S(n, m), i.e. S(n, m) mod 2.
<br>
<br>
<br>Example
<br><pre>
<br>S(4, 2) mod 2 = 1.
<br></pre>
<br>
<br>Task
<br>
<br>Write a program which for each data set:
<br>reads two positive integers n and m,
<br>computes S(n, m) mod 2,
<br>writes the result.
<br></div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of the input contains exactly one positive integer d equal to the number of data sets, 1 <= d <= 200. The data sets follow.
<br>
<br>Line i + 1 contains the i-th data set - exactly two integers ni and mi separated by a single space, 1 <= mi <= ni <= 10^9.
<br></div><p class="pst">Output</p><div class="ptx" lang="en-US">The output should consist of exactly d lines, one line for each data set. Line i, 1 <= i <= d, should contain 0 or 1, the value of S(ni, mi) mod 2.</div><p class="pst">Sample Input</p><pre class="sio">1
4 2</pre><p class="pst">Sample Output</p><pre class="sio">1</pre><p class="pst">

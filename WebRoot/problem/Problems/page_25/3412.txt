</p><div class="ptx" lang="en-US"><p>There is a structure consisting of <b>K</b> homogenous slices of equal thickness. The refraction index of the <b>i</b>-th slice is <b>N<sub>i</sub></b>. A ray of light travels in the first slice at the angle <b>a</b> to the normal. Write a program to compute the angle that the ray will have to the normal in the last slice.</p><p>It is known that the formula N<sub>1</sub>sin&#945;=N<sub>2</sub>sin&#946; holds in a two-sliced structure with the refraction indices of the slices equal to N<sub>1</sub> and N<sub>2</sub>, respectively, when a ray of light passes from the first slice to the second, traveling at the angle a to the normal in the first slice, and at the angle b to the normal in the second slice. </p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>On the first line of the file is the number of slices <b>K</b> (1&#8804;<b>K</b>&#8804;100). Each of the following <b>K</b> lines contains the refraction index <b>N<sub>i</sub></b> of the corresponding slice. All the refraction indices are real numbers in the range from 0.000001 to 2000. The last line of the file contains the angle <b>&#945;</b> between the ray and the normal in the first slice. The angle is a real number in the range from 0 to &#960;/2 and is given in radians.</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>The first and only line of the file must contain the angle <b>&#946;</b> between the ray and the normal in the last slice. The answer must be given with the precision of 0.001. If the ray does not reach the last slice, the output must be the word &#8216;<b>NO</b>&#8217;.</p></div><p class="pst">Sample Input</p><pre class="sio">2
1
2
0.02</pre><p class="pst">Sample Output</p><pre class="sio">0.01</pre><p class="pst">

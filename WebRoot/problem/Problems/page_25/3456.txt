</p><div class="ptx" lang="en-US"><p>The <i>Frobenius problem</i> is an old problem in mathematics, named after the German mathematician G. Frobenius (1849&#8211;1917).</p><p>Let <i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, &#8230;, <i>a<sub>n</sub></i> be integers larger than 1, with greatest common divisor (gcd) 1. Then it is known that there are finitely many integers larger than or equal to 0, that cannot be expressed as a linear combination <i>w</i><sub>1</sub><i>a</i><sub>1</sub> + <i>w</i><sub>2</sub><i>a</i><sub>2</sub> + &#8230; + <i>w<sub>n</sub>a<sub>n</sub></i> using integer coefficients <i>w<sub>i</sub></i> &#8805; 0. The largest of such nonnegative integers is known as the Frobenius number of <i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, &#8230;, <i>a<sub>n</sub></i> (denoted by <i>F</i>(<i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, &#8230;, <i>a<sub>n</sub></i>)). So: <i>F</i>(<i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, &#8230;, <i>a<sub>n</sub></i>) is the largest nonnegative integer that cannot be expressed as a nonnegative integer linear combination of <i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, &#8230;, <i>a<sub>n</sub></i>.</p><p>For <i>n</i> = 2 there is a simple formula for <i>F</i>(<i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>). However, for <i>n</i> &#8805; 3 it is much more complicated. For <i>n</i> = 3 only for some special choices of <i>a</i><sub>1</sub>, <i>a</i><sub>2</sub>, <i>a</i><sub>3</sub> formulas exist. For <i>n</i> &gt; 4 no formulas are known at all.</p><p>We will consider here the Frobenius problem for <i>n</i> = 4. In this case our version of the problem can be formulated as follows. Let four integers <i>a</i>, <i>b</i>, <i>c</i> and <i>d</i> be given, with <i>a</i>, <i>b</i>, <i>c</i>, <i>d</i> &gt; 1 and gcd(<i>a</i>, <i>b</i>, <i>c</i>, <i>d</i>) = 1. We want to know two things.</p><ul><li>How many nonnegative integers less than or equal to 1,000,000 cannot be expressed as a nonnegative integer linear combination of the values <i>a</i>, <i>b</i>, <i>c</i> and <i>d</i>?</li><li>Is the Frobenius number of <i>a</i>, <i>b</i>, <i>c</i> and <i>d</i> less than or equal to 1,000,000 and if so, what is its value?</li></ul></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The first line of the input file contains a single number: the number of test cases to follow. Each test case has the following format:</p><ul><li>One line, containing four integers <i>a</i>, <i>b</i>, <i>c</i>, <i>d</i> (with 1 &lt; <i>a</i>, <i>b</i>, <i>c</i>, <i>d</i> &#8804; 10, 000 and gcd(<i>a</i>, <i>b</i>, <i>c</i>, <i>d</i>) = 1), separated by single spaces.</li></ul></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>For every test case in the input file, the output should contain two lines.</p><ul><li>The first line contains the number of integers between 0 and 1,000,000 (boundaries included) that cannot be expressed as <i>a</i>&#8901;<i>w</i> + <i>b</i>&#8901;<i>x</i> + <i>c</i>&#8901;<i>y</i> + <i>d</i>&#8901;<i>z</i>, where <i>w</i>, <i>x</i>, <i>y</i>, <i>z</i> are nonnegative (meaning &#8805; 0) integers.</li><li>The second line contains the Frobenius number if this is less than or equal to 1,000,000 and otherwise &#8722;1, meaning that the Frobenius number of <i>a</i>, <i>b</i>, <i>c</i> and <i>d</i> is larger than 1,000,000.</li></ul></div><p class="pst">Sample Input</p><pre class="sio">3

</p><div class="ptx" lang="en-US"><p>There is a configuration of <b>N</b> domino stones that is to be beautifully destroyed. For each stone <b>i</b> in the given configuration, we know the set <b>D<sub>i</sub></b> of the stones that will fall at time <b>t</b>+1 (if they have not fallen before) if the stone <b>i</b> falls at time <b>t</b>. </p><p>Write a program to determine which stone should be pushed to fall at the time 0 to achieve complete destruction of the configuration that takes the longest possible time.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The first line of the input contains the number of stones <b>N</b> in the configuration (1&#8804;<b>N</b>&#8804;1000). Next <b>N</b> lines describe sets <b>D<sub>i</sub></b> (1&#8804;<b>i</b>&#8804;<b>N</b>). Each of these lines contains first the potency of the set <b>D<sub>i</sub></b> and then the numbers of stones in the set separated by one or more spaces.</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>The first line of the output must contain the time of fall of the last stone in the configuration, and the second line - the number of the stone that should be pushed first. If the maximal time can be achieved starting from several initial stones, output the maximal possible number.

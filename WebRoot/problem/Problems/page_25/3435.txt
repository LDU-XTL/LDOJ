</p><div class="ptx" lang="en-US"><p>The puzzle game of Sudoku is played on a board of <i>N</i><sup>2</sup> × <i>N</i><sup>2</sup> cells. The cells are grouped in <i>N</i> × <i>N</i> squares of <i>N</i> × <i>N</i> cells each. Each cell is either empty or contains a number between 1 and <i>N</i><sup>2</sup>.</p><p>The sudoku position is correct when numbers in each row, each column and each square are different. The goal of the game is, starting from some correct position, fill all empty cells so that the final position is still correct.</p><p>This game is fairly popular in the Internet, and there are many sites which allow visitors to solve puzzles online. Such sites always have a subroutine to determine a correctness of a given position.</p><p>You are to write such a routin.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>Input file contains integer <i>N</i>, followed by <i>N</i><sup>4</sup> integers — sudoku position.

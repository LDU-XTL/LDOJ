</p><div class="ptx" lang="en-US"><div><p>The repetition number of a string is defined as the maximum number <i>R</i> such that the string can be partitioned into <i>R</i> same consecutive substrings. For example, the repetition number of &quot;ababab&quot; is 3 and &quot;ababa&quot; is 1.</p><p>Given a string containing lowercase letters, you are to find a substring of it with maximum repetition number.</p></div></div><p class="pst">Input</p><div class="ptx" lang="en-US"><div><p>The input consists of multiple test cases. Each test case contains exactly one line, which<br>gives a non-empty string consisting of lowercase letters. The length of the string will not be greater than 100,000.</p><p>The last test case is followed by a line containing a '#'.</p></div></div><p class="pst">Output</p><div class="ptx" lang="en-US"><div><p>For each test case, print a line containing the test case number( beginning with 1) followed by the substring of maximum repetition number. If there are multiple substrings of maximum repetition number, print the lexicographically smallest one.</p></div></div><p class="pst">Sample Input</p><pre class="sio">ccabababc
daabbccaa
#</pre><p class="pst">Sample Output</p><pre class="sio">Case 1: ababab

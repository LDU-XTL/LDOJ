</p><div class="ptx" lang="en-US">The game of Tetris is played with four-connected blocks falling down the well having N rows and 20 columns. Each figure is marked with a unique English letter from 'A' to 'Z'. 
<br>Your program must, given the state of the well, determine the order in which the blocks fell down.</div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of input file contains integer N (1 <= N <= 50) -- number of rows. Following N lines contain 20 characters each, with characters that are either a letter from 'A' to 'Z' or the dot character (ASCII 46), representing an empty cell.</div><p class="pst">Output</p><div class="ptx" lang="en-US">Output file must contain a string of letters indicating the order in which figures fell down. If there is more than one order, lexicographically smallest one must be printed. Input data will guarantee that at least one nonempty order exists.</div><p class="pst">Sample Input</p><pre class="sio">6
...........XX.......
..........MMMM......
..........K.........
........KKK.........
.....ZAAA.FFF.......
.....ZZZA..F.B......
</pre><p class="pst">Sample Output</p><pre class="sio">BFZAKMX</pre><p class="pst">

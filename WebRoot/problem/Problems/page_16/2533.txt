</p><div class="ptx" lang="en-US">A numeric sequence of <I>a<SUB>i</SUB></I> is ordered if <I>a<SUB>1</SUB></I> < <I>a<SUB>2</SUB></I> < ... < <I>a<SUB>N</SUB></I>. Let the subsequence of the given numeric sequence (<I>a<SUB>1</SUB></I>, <I>a<SUB>2</SUB></I>, ..., <I>a<SUB>N</SUB></I>) be any sequence (<I>a<SUB>i<SUB>1</SUB></SUB></I>, <I>a<SUB>i<SUB>2</SUB></SUB></I>, ..., <I>a<SUB>i<SUB>K</SUB></SUB></I>), where 1 <= <I>i<SUB>1</SUB></I> < <I>i<SUB>2</SUB></I> < ... < <I>i<SUB>K</SUB></I> <= <I>N</I>. All longest ordered subsequences are of length 4, e. g., (1, 3, 5, 8).<BR><BR>Your program, when given the numeric sequence, must find the length of its longest ordered subsequence.</div><p class="pst">Input</p><div class="ptx" lang="en-US">The first line of input file contains the length of sequence N.

</p><div class="ptx" lang="en-US"><span lang="en-us"><p>A two-player normal form game between two individuals A and B is completely specified by</p><ul><li>{<i>a</i><sub>1</sub>, &hellip;, <i>a<sub>m</sub></i>}, a set of actions for player A,</li><li>{<i>b</i><sub>1</sub>, &hellip;, <i>b<sub>n</sub></i>}, a set of actions for player B,</li><li><i>PA</i>, an <i>m</i> &times; <i>n</i> payoff matrix for player A, and</li><li><i>PB</i>, an <i>m</i> &times; <i>n</i> payoff matrix for player B.</li></ul><p>In such a game, both players simultaneously select actions to be played (say <i>a<sub>i</sub></i> and <i>b<sub>j</sub></i> for players A and B, respectively). Then payoffs for each player are determined according to the payoff matrices (<i>PA</i>[<i>i</i>, <i>j</i>] and <i>PB</i>[<i>i</i>, <i>j</i>] for players A and B, respectively). The goal of each player is to maximize his or her payoff.</p><p>For player A, the set of <i>best responses</i> to a particular action <i>b<sub>j</sub></i> by player B consists of any action <i>a<sub>i</sub></i> which maximizes Aâ€™s payoff, that is, whose payoff is <i>max<sub>i'</sub></i> <i>PA</i>[<i>i'</i>, <i>j</i>]. Similarly, for player B, the set of best responses to a particular action <i>a<sub>i</sub></i> by player A is any action <i>b<sub>j</sub></i> whose payoff is <i>max<sub>j'</sub></i> <i>PB</i>[<i>i</i>, <i>j'</i>]. <i>b<sub>j</sub></i> is a best response to <i>a<sub>i</sub></i>.</p><p>In this problem, you are given the payoff matrices for two players A and B, and your task is to find and list all pure strategy

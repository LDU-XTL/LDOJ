</p><div class="ptx" lang="en-US"><span lang="en-us"><p>There are <b>N</b> points marked on a surface, pair (<b>x<sub>i</sub></b>, <b>y<sub>i</sub></b>) is coordinates of a point number <b>i</b>. Let&rsquo;s call a <b>necklace</b> a set of <b>N</b> figures which fulfills the following rules.</p><ul><li>The figure #<b>i</b> consists of all such points (<b>x</b>, <b>y</b>) that (<b>x</b> &minus; <b>x<sub>i</sub></b>)<sup>2</sup> + (<b>y</b> &minus; <b>y<sub>i</sub></b>)<sup>2</sup> &le; <b>r<sub>i</sub></b><sup>2</sup>, where <b>r<sub>i</sub></b> &ge; 0.</li><li>Figures #<b>i</b> and #(<b>i</b> + <b>1</b>) intersect (1 &le; <b>i</b> &lt; <b>N</b>)</li><li>Figures #<b>1</b> and #<b>N</b> intersect</li><li>All the rest pairs of figures do not intersect.</li></ul><p>Write a program which takes points and constructs a necklace.</p></span></div><p class="pst">Input</p><div class="ptx" lang="en-US"><span lang="en-us"><p>The first line of the input contains one integer number <b>N</b> (2 &le; <b>N</b> &le; 100). Each of the next <b>N</b> lines contains two real numbers <b>x<sub>i</sub></b>, <b>y<sub>i</sub></b> (1 000 &le; <b>x<sub>i</sub></b>, <b>y<sub>i</sub></b> &le; 1 000), separated by one space. It is guaranteed that at least one necklace exists.</p></span></div><p class="pst">Output</p><div class="ptx" lang="en-US"><span lang="en-us"><p>The output has to contain <b>N</b> lines.

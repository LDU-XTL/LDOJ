</p><div class="ptx" lang="en-US"><p>In a speech contest, when a contestant finishes his speech, the judges will then grade his performance. The staff remove the highest grade and the lowest grade and compute the average of the rest as the contestant&rsquo;s final grade. This is an easy problem because usually there are only several judges.</p><p>Let&rsquo;s consider a generalized form of the problem above. Given <i>n</i> positive integers, remove the greatest <i>n</i><sub>1</sub> ones and the least <i>n</i><sub>2</sub> ones, and compute the average of the rest.</p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The input consists of several test cases. Each test case consists two lines. The first line contains three integers <i>n</i><sub>1</sub>, <i>n</i><sub>2</sub> and <i>n</i> (1 &le; <i>n</i><sub>1</sub>, <i>n</i><sub>2</sub> &le; 10, <i>n</i><sub>1</sub> + <i>n</i><sub>2</sub> &lt; <i>n</i> &le; 5,000,000) separate by a single space. The second line contains <i>n</i> positive integers <i>a<sub>i</sub></i> (1 &le; <i>a<sub>i</sub></i> &le; 10<sup>8</sup> for all <i>i</i> s.t. 1 &le; <i>i</i> &le; <i>n</i>) separated by a single space. The last test case is followed by three zeroes.</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>For each test case, output the average rounded to six digits after decimal point in a separate line.</p></div><p class="pst">Sample Input</p><pre class="sio">1 2 5
1 2 3 4 5
4 2 10
2121187 902 485 531 843 582 652 926 220 155
0 0 0</pre><p class="pst">Sample Output</p><pre class="sio">3.500000
562.500000</pre><p class="pst">Hint</p><div class="ptx" lang="en-US"><p>This problem has very large input data. <i>scanf</i> and <i>printf</i> are recommended for C++ I/O.</p><p>The memory limit might not allow you to store everything in the memory.</p></div><p class="pst">

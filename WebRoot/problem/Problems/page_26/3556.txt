</p><div class="ptx" lang="en-US"><p>A sequence <b>p(1)</b>, <b>p(2)</b>, …, <b>p(N)</b> consisting of numbers 1, 2, …, <b>N</b> is called <i>a</i> <i>permutation </i>if all elements in the sequence are different.</p><p>It is said that a permutation <b>p</b> contains <i>increasing subsequence</i> of <b>k </b>elements when there are numbers 1 ≤ <b>i<sub>1</sub></b> &lt; <b>i<sub>2</sub></b> &lt; … &lt; <b>i<sub>k</sub></b> ≤ N such that <b>p(i<sub>1</sub>)</b> &lt; <b>p(i<sub>2</sub>)</b> &lt; … &lt; <b>p(i<sub>k</sub>)</b>.</p><p>When a permutation <b>p</b> contains an <i>increasing subsequence</i> consisting of <b>B </b>elements and does not contain an <i>increasing subsequence<b> </b></i>consisting of <b>B+1</b> elements then the number <b>B </b>is called <i>the degree of increase </i>of this permutation.</p><p>You need to write a program which being given a number <b>N</b> calculates the number of permutations whose <i>degree of increase </i>is <b>B</b>. Since the number of such permutations might be quite big, it is necessary to calculate its remainder of integer division by <nobr>1 000 000 000.</nobr></p></div><p class="pst">Input</p><div class="ptx" lang="en-US"><p>The input consists of one line. The line contains two integer numbers <b>N</b> and <b>B</b> (1 ≤ <b>N</b> ≤ 40, 1 ≤ <b>B</b> ≤ 5) separated by one or more spaces.</p></div><p class="pst">Output</p><div class="ptx" lang="en-US"><p>The output contains one integer number which is the remainder of integer division by <nobr>1 000 000 000</nobr> of the number of permutations whose <i>degree of increase </i>is <b>B</b>.</p></div><p class="pst">Sample Input</p><pre class="sio">3 2</pre><p class="pst">Sample Output</p><pre class="sio">4</pre><p class="pst">
